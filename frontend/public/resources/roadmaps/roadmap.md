# TechLearn Roadmap

This roadmap gives a practical path from fundamentals to interview readiness.

## Phase 1: Core Foundations (Weeks 1-3)

- Learn one language deeply: C, Java, or Python
- Practice control flow, functions, arrays, and strings
- Build 3 small console projects

## Phase 2: DSA Essentials (Weeks 4-8)

- Arrays and strings patterns
- Linked lists, stacks, and queues
- Recursion and binary search
- Trees and heaps basics
- Hashing and sliding window

## Phase 3: SQL and Backend Basics (Weeks 9-11)

- SQL CRUD, joins, grouping, and subqueries
- Build a small Express + MongoDB API
- Add auth and validation

## Phase 4: Interview Preparation (Weeks 12-14)

- Solve 2-3 coding questions per day
- Revisit weak topics every weekend
- Practice company-based and core CS questions
- Take mock interviews

## Weekly Routine

1. 90 minutes problem solving
2. 45 minutes revision
3. 30 minutes notes and recap

## Outcome

By the end of this roadmap, you should be ready for internship or entry-level interview rounds.
